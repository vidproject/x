GitHub connection

Paste a fine-grained Personal Access Token scoped to a single repo, with Contents: Read & Write and Metadata: Read. Generate one at github.com/settings/personal-access-tokens/new.

The token is stored in browser.storage.local and is readable by anyone with filesystem access to your Firefox profile. Never use a classic repo-scoped token here.

Tracked accounts

Loaded from config/accounts.yaml in the configured repo. Edit the YAML and push to add or remove handles — the extension re-reads it on startup.

    Diagnostics

    Copy the diagnostic dump below when reporting a problem. It contains your configured owner/repo, the last connection result, recent activity, and a redacted indicator that a PAT is set (the token itself is never included).
